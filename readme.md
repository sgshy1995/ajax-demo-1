### AJAX 的实现

使用 AJAX 实现前后端互动。

#### 功能

- 用 AJAX 加载 CSS
- 用 AJAX 加载 JS
- 用 AJAX 加载 HTML，出现在页面
- 用 AJAX 加载 XML，获取节点内容并出现在页面
- 用 AJAX 加载 JSON，转为对象并转为列表展示在页面
- 用 AJAX 模拟分页操作，获取数据库内容展示在页面

#### 开始

```bash
$ node server.js 8888
```

你也可以使用 node-dev 工具用于开发（实时重启服务端）

```bash
$ yarn global add node-dev
$ node-dev server.js 8888
```
